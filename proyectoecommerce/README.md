# Proyecto Final Firestore/Vue

## Setup del proyecto
```
npm install
```
### instalar firebase para la BD
```
npm install --save firebase
```
### Compilación y actualización automática para desarrollo
```
npm run serve
```

### Compilación y compresión 
```
npm run build
```

### Correr tests
```
npm run test
```

### Solucionar con Lint
```
npm run lint
```
